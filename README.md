# Adrita Idris Disha Portfolio Website

This is a customized one-page portfolio website based on the supplied Thomson template. The content has been rewritten from the provided CV into a public-facing portfolio format.

## How to open locally

Open `index.html` in a browser.

## How to deploy

Upload the full folder to any static host, such as GitHub Pages, Netlify, Vercel, or cPanel hosting. Keep the folder structure unchanged so CSS, JavaScript, images, and the CV link continue to work.

## Main editable files

- `index.html` - website content and sections
- `css/style.css` - visual styling
- `images/adrita-profile.jpg` - profile image extracted from the CV
- `resume/Adrita-Idris-Disha-CV.pdf` - downloadable CV

## Privacy note

The webpage avoids displaying date of birth, gender, and phone number. The downloadable CV still contains the details present in the uploaded CV.
